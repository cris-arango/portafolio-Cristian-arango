## Descripción del proyecto
Portafolio personal desarrollado con HTML, CSS y PHP.  
Incluye:
- Página principal con información personal, foto y hobbies
- Formulario de contacto completamente funcional
- Los mensajes se guardan en una base de datos MySQL
- Validación de campos obligatorios y correo electrónico
- Redirección a página de confirmación al enviar el mensaje
- Diseño responsive y moderno

## Tecnologías utilizadas
- HTML5
- CSS3 (diseño personalizado)
- PHP  PDO (procesamiento seguro del formulario)
- MySQL (base de datos para almacenar mensajes)
- Hosting gratuito: InfinityFree
